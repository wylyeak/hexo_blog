hexo_blog
=========
## hexo init
see [INSTALL][1]

1. 安装[nodejs][2]
2. 安装[Git][3]
3. 执行安装
```shell
npm install -g hexo
```
## blog init
``` shell
git clone https://github.com/wylyeak/hexo_blog.git
cd hexo_blog
sudo npm install
sudo npm install hexo-generator-feed
sudo npm install hexo-generator-sitemap
git clone https://github.com/wylyeak/hexo-theme-yilia.git themes/yilia
```
