hexo_blog
=========
## hexo init
see [INSTALL][1]

1. 安装[nodejs][2]
2. 安装[Git][3]
3. 执行安装
```shell
npm install -g hexo
```
## blog init
``` shell
git clone --recursive git@github.com:wylyeak/hexo_blog.git
cd hexo_blog
npm install
npm install hexo-generator-feed
npm install hexo-generator-sitemap
```
